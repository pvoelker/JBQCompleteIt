Simple quizzing game application for kids based on 20-point Junior Bible Quizzing questions. A 'Learning Mode' helps them intially learn the questions. Three different difficulties are available: Easy, Normal and Hard

If a wrong answer is provided, the incorrect parts are reset. And you keep trying until you get it right.

Animations for correct answers help keep the kids engaged.

See the Downloads section on links to get the application.

A game for 10-point questions is available and is called JBQ Quiz Me!.

If you are interested in Junior Bible Quizzing, checkout: https://kidmin.ag.org/ministries/JBQ/overview
